# React Date Counter

A simple React app to increment or decrement a date by a chosen step and display the result.

## Features

- Increment/decrement the step value
- Increment/decrement the date by the current step
- Shows how many days from/till today and the resulting date

## Installation

1. Make sure you have [Node.js](https://nodejs.org/) and npm installed.
2. Install dependencies:
   ```
   npm install
   ```

## Usage

Start the development server:

```
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the app in your browser.

Use the buttons to change the step and count. The message updates with the new date and how many days from/till today.

## Folder Structure

- `src/` - React components and logic
- `public/` - Static files
